# 已知殘留問題

Phase 1 分級修正後仍存在的三個已知問題，記錄現象、影響、暫不處理的理由。之後若要動，先讀這份文件避免重複踩坑。

## 1. 片假名外來語 fallback 一律估為 L3，可能高估一級

**現象**：`grade.ts` 的三段 fallback 中，片假名外來語只要 jmdict 有詞條就一律給 L3（N3），不分實際難度。例如「パン」（麵包，日常基礎詞彙）與較生僻的外來語會拿到相同的 L3，儘管前者更接近 N5。

**影響**：包含常見基礎外來語（パン、バス、テレビ 等）的簡單句子，難度可能被高估一級，例如原本該落在 N4 的句子被判成 N3。由於 90th percentile 只取一小撮最高值，短句若剛好含一個這類詞，影響會比長句明顯。

**暫不處理理由**：要精確分級外來語需要額外資料源（例如判斷該詞是否也出現在 JLPT 官方或坊間片假名詞表），現有輸入（JLPT 單字表 + jmdict）沒有片假名詞彙的分級資訊，L3 是「不算最難、但也不算查無」的折衷值。使用者已明確指示這次不改 percentile 主邏輯與不做進一步演算法微調，留待之後有更好的片假名分級資料源時再處理。

## 2. 形式名詞「の」等被當實詞計入

**現象**：kuromoji 將形式名詞（の、こと、もの 用作準體助詞時）標為「名詞,非自立」，這次的排除規則只涵蓋「動詞,接尾／非自立」與「名詞,数／記号」，沒有涵蓋「名詞,非自立」，所以「の」這類語法功能詞仍會被算進實詞、並透過 jmdict fallback 拿到 L3。

**影響**：例如「〜のは苦痛だ」這種常見句型，「の」本身不該算難度來源，卻貢獻了一個 L3 樣本，可能把原本該更低的句子難度往上推。

**暫不處理理由**：這次驗收的三個修正目標（れる/られる/せる/させる、動詞接尾/非自立、名詞數字/符號）已明確指定範圍，「名詞,非自立」的排除牽涉更廣（こと、もの、はず、わけ 等一整組形式名詞，其中部分用法仍應算實詞，例如「もの」單獨使用表示「東西」時），需要更細緻的規則設計，故列為已知問題留待下次一併處理，不在本次範圍內臨時加規則。

## 3. N2／N1 單字例句覆蓋率偏低（65.8%／68.9%）

**現象**：N5–N3 單字有例句的比例落在 81–90%，但 N2 是 65.8%、N1 是 68.9%，明顯偏低。

**影響**：約三分之一的 N1/N2 單字目前沒有任何例句，複習卡正面只有單字本身、翻面缺例句佐證用法。

**暫不處理理由**：這是 tatoeba-json 例句索引資料本身的覆蓋率限制——較高階單字在日常例句語料庫中出現頻率本來就低，不是連結邏輯的 bug。要提升覆蓋率需要额外例句來源或放寬比對條件（例如允許非 dictionary-form 的部分比對），兩者都超出本次修正範圍，且可能引入誤配對風險，留待之後評估是否需要額外資料源。
